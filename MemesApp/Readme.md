## Memes App
This is a simple begineer friendly Memes app which fetched random Meme from reddit.

### Technologies/ Libraries Uesd 
- Android
- Kotlin
- API calls
- Volley Library
- Glide Library

### Run and Build
To run and build this app you would need Android studio and knowledge of Kotlin language
