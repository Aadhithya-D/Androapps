# MoviesX
### A Movie searching app build to demonstrate Modern Android Devleopment using Kotlin 

## Technology Used
 1. Android
 2. Kotlin

 ## Libraries and APIs Used
 - Retrofit
 - Gson Converter
 - Coroutines And Flow
 - OMDB REST Api
 - ViewBinding
 - Dagger Hilt
 
### Requirements
1. Android Version 5.0 and above
2. CompileSdkVersion 31
3. MinSdkVersion 21
4. targetSdkVersion 31
5. Androidx
